# .dotfiles

Version control for my dotfiles

## Setup

1. Install [stow](https://www.gnu.org/software/stow/)
2. Install dotfiles

```sh
cd $HOME/.dotfiles
chmod +x ./install.sh
./install.sh
```

Still have to manually install these with package manager.
